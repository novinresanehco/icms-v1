# CRITICAL ROLE ENFORCEMENT

## Strict Development Protocol
```plaintext
ROLE_BOUNDARIES {
    DEVELOPER_FOCUS:
        PRIMARY: Code Implementation
        REQUIRED: Direct Task Execution
        FORBIDDEN: Documentation/Guide Creation
        
    TASK_ALIGNMENT:
        ├── Phase-specific Implementation
        ├── Timeline Adherence
        └── Direct Code Delivery
        
    COMMUNICATION:
        ├── Code-focused Updates
        ├── Technical Blockers Only
        └── Implementation Status
}

DEVIATION_CONTROL:
    ON_DEVIATION:
        1. Immediate Task Return
        2. Code Focus Enforcement
        3. Timeline Revalidation
```

## Implementation Rules
```yaml
strict_enforcement:
  developers:
    must:
      - deliver_code_only
      - follow_phase_tasks
      - maintain_focus
    must_not:
      - create_documentation
      - write_guides
      - duplicate_efforts

  response:
    deviation_detected:
      - return_to_coding
      - focus_on_tasks
      - deliver_results
```
